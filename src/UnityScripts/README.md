# Unity Scripts

This folder contains selected C# scripts from the TORO Unity project.

## Included Files

- `TutorialStart.cs`: controller/raycast tutorial UI dismissal
- `HitRayHand.cs`: raycast-based lantern/candle lighting interaction prototype
- `BasketInteraction.cs`: trigger-based collection, UI count update, and spawned flower feedback
- `PickupDebris.cs`: raycast-based collectible targeting prototype
- `PlayerMovement.cs`: desktop movement prototype
- `FirstPersonControl.cs`: desktop first-person camera movement prototype

## Context

These scripts are shown alongside the technical breakdown to document my Unity/C# implementation work. They depend on scene objects, prefabs, input action assets, and Unity editor configuration from the original project.
